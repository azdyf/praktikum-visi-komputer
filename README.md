# Praktikum Visi Komputer

Repository ini berisi seluruh hasil praktikum mata kuliah **Visi Komputer**.  
Setiap folder merepresentasikan satu atau beberapa pertemuan praktikum dan berisi implementasi kode Python sesuai dengan modul yang diberikan.

## Tujuan
Repository ini dibuat untuk:
- Mendokumentasikan hasil praktikum secara terstruktur
- Menyimpan implementasi kode Python setiap pertemuan
- Menjadi referensi pembelajaran visi komputer

## Tools dan Library
- Python 3
- NumPy
- Pandas
- Matplotlib
- OpenCV (`opencv-contrib-python`)

## Keterangan
- Setiap folder `LKP-X` berisi:
  - Source code Python
  - Dataset (jika ada)
  - README penjelasan per pertemuan
- Repository ini akan terus diperbarui sesuai dengan perkembangan materi praktikum.
